Perguntas frequentes

Dúvidas respondidas.

Preciso saber de marketing ou IA para usar?
+
Não. O sistema foi desenvolvido para quem não tem esse conhecimento. Você responde um formulário sobre seu negócio uma única vez e os agentes fazem o resto. Você apenas aprova o conteúdo antes de publicar.
Qual a diferença entre Básico e Plus?
+
No Básico você baixa os criativos e agenda manualmente. No Plus tudo é publicado automaticamente após sua aprovação — você clica "Aprovar" e o sistema cuida do resto. Ambos têm os mesmos 5 agentes de criação.
Como funcionam os 7 dias de garantia?
+
Após o cadastro, você tem 7 dias para testar sem qualquer cobrança. Se cancelar dentro do prazo, nada é cobrado. Direito garantido pelo Código de Defesa do Consumidor (Art. 49 do CDC).
Funciona para qualquer segmento de negócio?
+
Sim. O sistema é universal — cada cliente tem sua própria identidade e estratégia geradas do zero com base no check-in. Funciona para consultores, tatuadores, confeiteiras, corretores, personal trainers, e-commerce e outros.
O que é o Material Didático JUMP?
+
É o método completo em formato de guia digital. Com ele você consegue executar toda a estratégia manualmente usando IAs gratuitas como o ChatGPT. Os agentes são o atalho que automatiza o que o material ensina. Incluído em todos os planos.
