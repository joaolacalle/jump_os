Política de Privacidade

Última atualização: 1 de junho de 2026 · Em conformidade com a LGPD (Lei nº 13.709/2018)

Esta política descreve como coletamos, usamos e protegemos seus dados pessoais. Levamos sua privacidade a sério e operamos em conformidade com a Lei Geral de Proteção de Dados (LGPD).

1. Dados que Coletamos

Dados de cadastro:

  • Nome completo, e-mail, telefone/WhatsApp
  • CPF (apenas para emissão de nota fiscal)
  • Dados de pagamento (processados pelo Stripe — não armazenamos cartão)

Dados do negócio (fornecidos no check-in):

  • Informações sobre seu negócio, público-alvo e objetivos
  • Logotipo, fotos e materiais de marca enviados voluntariamente

Dados das redes sociais (mediante autorização):

  • Métricas públicas do Instagram (alcance, engajamento)
  • Acesso para publicação (somente após sua autorização via OAuth)

Dados de uso do sistema:

  • Logs de acesso, ações na plataforma, conteúdos aprovados/rejeitados
  • Preferências e configurações de conta

2. Como Usamos seus Dados

  • Para personalizar a estratégia e conteúdo gerado pelos agentes de IA
  • Para processar pagamentos e gerenciar sua assinatura
  • Para enviar comunicações relacionadas ao serviço (transacionais)
  • Para melhorar o sistema com dados agregados e anonimizados
  • Para cumprir obrigações legais e fiscais

3. Compartilhamento de Dados

Seus dados pessoais identificáveis nunca são vendidos. Compartilhamos apenas com:

  • Stripe: processamento de pagamentos (certificação PCI-DSS)
  • Supabase: armazenamento seguro de dados (servidores na UE/EUA)
  • Anthropic / Leonardo AI: processamento de IA (dados anonimizados)
  • Meta/Instagram: publicação automática (somente com sua autorização)
  • Resend: envio de e-mails transacionais

Todos os fornecedores são submetidos a avaliação de conformidade com LGPD/GDPR.

4. Seus Direitos (LGPD Art. 18)

  • Confirmação e acesso: saber quais dados temos sobre você
  • Correção: atualizar dados incompletos ou incorretos
  • Anonimização ou exclusão: solicitar remoção de dados desnecessários
  • Portabilidade: exportar seus dados em formato estruturado
  • Revogação do consentimento: retirar permissões a qualquer momento
  • Oposição: se opor ao tratamento em casos previstos na lei

Para exercer qualquer um desses direitos, envie e-mail para: privacidade@jumpos.com.br

5. Retenção de Dados

  • Dados de conta ativa: mantidos enquanto a assinatura estiver ativa
  • Após cancelamento: dados mantidos por 90 dias para eventual reativação
  • Após 90 dias: exclusão definitiva mediante solicitação ou automaticamente
  • Logs do sistema: excluídos automaticamente após 90 dias
  • Dados fiscais: mantidos pelo prazo legal de 5 anos (obrigação tributária)

6. Segurança

  • Transmissão criptografada via HTTPS/TLS em todas as comunicações
  • Senhas armazenadas com hash bcrypt (nunca em texto claro)
  • Acesso ao banco de dados restrito por Row Level Security (RLS)
  • Tokens de acesso ao Instagram armazenados criptografados
  • Autenticação de dois fatores disponível para conta admin

7. Cookies

Usamos apenas cookies estritamente necessários para manter sua sessão logada. Não usamos cookies de rastreamento de terceiros para fins publicitários.

8. Menores de Idade

O JUMP OS não é destinado a menores de 18 anos. Não coletamos intencionalmente dados de crianças ou adolescentes. Se identificarmos tal situação, a conta será encerrada e os dados excluídos.
